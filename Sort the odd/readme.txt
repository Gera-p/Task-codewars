У вас есть массив чисел.
Ваша задача сортировать возрастающие нечетные числа, но четные числа должны быть на своих местах.
Ноль не является странным числом, и вам не нужно его перемещать. Если у вас есть пустой массив, вам нужно его вернуть.
пример:
sortArray ([5, 3, 2, 8, 1, 4]) == [1, 3, 2, 8, 5, 4]
